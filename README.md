# Simple Twitter Clone

A lightweight **Twitter-like feed application** built with **HTML, CSS, and JavaScript**.  
Live demo: [Simple Twitter Clone](https://simple-twitter-clone.netlify.app)

---

## Features
- Dynamic tweet feed rendered from JavaScript data objects
- Like, retweet, and reply interactions with instant updates
- Add new tweets via input field
- Toggle replies visibility
- Responsive layout styled with CSS
- Static profile pictures and icons served from `/public/images`

---

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Deployment**: [Netlify](https://www.netlify.com/)

---

## Learning Goals
- Practice DOM manipulation and event handling
- Manage state in a simple frontend app
- Configure Vite and deploy to Netlify
= Serve static assets correctly in modern build tools
